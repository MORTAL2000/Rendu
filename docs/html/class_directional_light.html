Detailed Description

A directional light, where all light rays have the same direction.

It can be associated with a shadow 2D map with orthogonal projection, generated using Variance shadow mapping. It is rendered as a fullscreen squad in deferred rendering.

See also
GPU::Frag::Directional_light, GPU::Frag::Light_shadow, GPU::Frag::Light_debug
